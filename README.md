# 🧵 RepareBem — Simulado de Prova (Java)

Implementação de um sistema simples em Java para gerenciamento de reparos em roupas e controle de ordens de serviço de uma loja fictícia chamada **RepareBem**.
O sistema permite cadastrar reparos, criar ordens de serviço, adicionar reparos a uma ordem, alterar o status das ordens e calcular o valor total dos serviços.

Desenvolvido como **simulado de prova** para a disciplina **Programação 2** da **Universidade Federal de Campina Grande (UFCG)**.
